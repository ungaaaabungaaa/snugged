# Environment Variables

Copy each `.env.example` before running locally. Never commit real secrets.

## Site and Medusa

- `NEXT_PUBLIC_SITE_URL`
- `MEDUSA_BACKEND_URL`
- `NEXT_PUBLIC_MEDUSA_BACKEND_URL`
- `NEXT_PUBLIC_MEDUSA_PUBLISHABLE_KEY`
- `DATABASE_URL`
- `REDIS_URL`
- `MEILISEARCH_HOST`
- `MEILISEARCH_API_KEY`
- `JWT_SECRET`
- `COOKIE_SECRET`
- `STORE_CORS`
- `ADMIN_CORS`
- `AUTH_CORS`

## Cloudflare R2

- `R2_ACCOUNT_ID`
- `R2_BUCKET`
- `R2_ACCESS_KEY_ID`
- `R2_SECRET_ACCESS_KEY`
- `R2_ENDPOINT`
- `R2_PUBLIC_BASE_URL`

## WhatsApp and OTP

- `INTERNAL_API_KEY`
- `OPENWA_API_URL`
- `OPENWA_API_KEY`
- `WHATSAPP_SESSION_ID`
- `WHATSAPP_BUSINESS_NUMBER`
- `OTP_MODE`
- `OTP_TTL_SECONDS`

Use `OTP_MODE=stub` until the WhatsApp session is ready. Stub verification accepts `000000`.

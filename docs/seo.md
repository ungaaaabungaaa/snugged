# SEO

SEO is scaffolded from Day 1.

Implemented surfaces:

- Next.js metadata API in the root layout and product pages.
- Product JSON-LD.
- Breadcrumb JSON-LD.
- Organization JSON-LD.
- `robots.ts`.
- `sitemap.ts`.
- `/llms.txt`.
- SEO landing pages:
  - `/thrift-clothes-india`
  - `/thrift-clothes-bengaluru`
  - `/ai-try-on-clothes`
  - `/one-of-one-fashion`
  - `/affordable-fashion-india`

Product URL format:

```txt
/product/[handle]
```

Category URL format:

```txt
/shop/[category]
```

Keep cart, checkout, wishlist, and account pages out of the index.
